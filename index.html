New macOS Login Experience

Microsoft Authentication with MFA

Overview

We're updating the macOS login experience to use Microsoft's authentication with Multi-Factor Authentication (MFA). This change enhances security while providing a familiar login experience.

What's Changing?

  • You'll now see a Microsoft login screen when starting your Mac
  • Enter your Microsoft email address to begin the login process
  • Complete MFA verification using your preferred method (Authenticator app, SMS, etc.)
  • Access your Mac once authentication is complete

Need Help?

If you encounter any issues with the new login process, please contact IT Support.
